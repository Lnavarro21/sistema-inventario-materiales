Acerca del Sistema

Sistema de Gestión de Inventario v3.0 (Edición Corporativa)

Software administrativo desarrollado exclusivamente para el Área de Sistemas del Rectorado de la USMP. Su objetivo principal es asegurar la integridad, trazabilidad y control estricto del stock operativo de materiales y herramientas, previniendo el desabastecimiento.

Stack Tecnológico
  • Frontend: Interfaz SPA, Bootstrap 5, Chart.js
  • Animaciones: GSAP Engine (Transición Cinematográfica)
  • Backend: Node.js, Express API REST
  • Base de Datos: SQLite3 Relacional
  • Seguridad: Autenticación Hash Bcrypt + SessionStorage
